# Discord Bot

The Discord bot is intentionally **out of v1 scope** for the `mcbanners-api-next` Bun rewrite.

This directory contains a stub placeholder only. It is not included in the monorepo build or typecheck references (`tsconfig.build.json`). It is not deployed as part of the v1 API container.

The legacy `../discord-api` Java Discord bot continues to operate separately.

See `docs/migration/027-bannertype-coverage.md` for the `DISCORD_USER` banner type coverage decision.
